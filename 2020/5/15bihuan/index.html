10-排序6 Sort with Swap(0, i)

给定N个序列，每次只使用Swap(0, *)实现排序

Posted by 徐强强 on Saturday, May 16, 2020

题目来自—中国大学MOOC-陈越、何钦铭-数据结构-2020春

Given any permutation of the numbers {0, 1, 2,…, N−1}, it is easy to sort them in increasing order. But what if Swap(0, *) is the ONLY operation that is allowed to use? For example, to sort {4, 0, 2, 1, 3} we may apply the swap operations in the following way: Swap(0, 1) => {4, 1, 2, 0, 3} Swap(0, 3) => {4, 1, 2, 3, 0} Swap(0, 4) => {0, 1, 2, 3, 4}

Now you are asked to find the minimum number of swaps need to sort the given permutation of the first N nonnegative integers. Input Specification: Each input file contains one test case, which gives a positive N (≤10) followed by a permutation sequence of {0, 1, …, N−1}. All the numbers in a line are separated by a space. Output Specification: For each case, simply print in a line the minimum number of swaps need to sort the given permutation.

Sample Input: 10 3 5 7 2 6 4 9 0 8 1

Sample Output: 9

解题思路

思考N个序列里有多少个环，并且环的种类。

单环不需要和0交换，其它的环思考是否有0元素；

有0元素的只需交换环中元素-1；

环中没有0元素，则环中任意一个元素先要和0交换，总共交换次数为环中元素个数。

代码（C语言实现）

#include<stdio.h>

#include<malloc.h>

int a[100000];

int main()

{

int N,i,j;

int ring = 1;

int count = 0;

int flag = 0;

scanf("%d\n",&N);

for(i=0;i<N;i++){//输入a[]

    scanf("%d",&a[i]);
	
}

for(i=0;i<N;i++)

{

    if( a[i] != i )
	
    {
	
        count++;
		
    }
	
}

if( a[0] == 0 )

    count++;
	
while(1)

{

    if( a[flag] != flag )
	
    {
	
        j = a[flag];
		
        a[flag] = flag;
		
        flag = j;
		
    }
	
    else
	
    {
	
        for(i=0;i<N;i++)
		
        {
		
            if(a[i] != i)
			
            {
			
                flag = i;
				
                ring++;
				
                break;
				
            }
			
        }
		
        if( i == N )
		
            break;
			
    }
	
}

//printf("%d %d\n", count, ring);

printf("%d\n", count+ring-2);

return 0;

}
