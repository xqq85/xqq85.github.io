Linux软件安装

Linux上安装软件常用方式

二进制发布包

软件已经针对具体平台编译打包发布，只要解压，修改配置即可

RPM包

软件已经按照redhat的包管理工具规范RPM进行打包发布，需要获得相应的软件RPM发布包，然后用RPM命令进行安装

Yum在线安装

软件已经以RPM规范打包，但发布在了网络上的一些服务器上，可用yum在线安装服务器山歌rpm软件，并且会自动解决软件安装过程的库依赖问题

源码编译安装

软件以源码工程的形式发布，需要获得源码工程后用相应的开发工具进行编译打包部署

上传与下载工具

FileZilla

lrzsz

需要在Linux上安装   yum   install   lrzsz

按ALT+P

使用   put   文件位置   命令上传文件（文件在Linux当前用户的操作目录里）
使用   get   文件   命令下载文件（下载到电脑文档目录）

安装jdk

Linux默认安装了jdk，需要先卸载，再安装自己的jdk版本

卸载open-JDK

查看jdk版本

java   -version

查看安装的JDK信息

rpm   -qa   |   grep   java

卸载jdk

rpm   -e   –nodeps   java-1.6.0-openjdk-1.6.0.35-1.13.7.1.e16_6.1686（JDK版本信息）

安装JDK

通常将软件安装到/usr/local
直接解压就可以
tar   -zxvf   jdk.tar.gz   -C   目标路径

配置环境变量

vi /etc/profile
在末尾行添加
#set   java   environment
JAVA_HOME=/usr/local/jdk/jdk1.7.0_71(解压文件夹的名字)
CLASSPATH=.:$JAVA_HOME/lib.tools.jar
PATH=$JAVA_HOME/bin:$PATH
export   JAVA_HOME   CLASSPATH   PATH

重新加载文件/etc/profile

source /etc/profile

查看JAVA安装情况

java   -version

安装Mysql

Linux默认安装了Mysql，需要先卸载，再安装自己的Mysql版本

将mysql的安装文件上传到Linux的服务器

卸载Mysql

查看安装的Mysql信息

rpm   -qa   |   grep   mysql

卸载mysql

rpm   -e   –nodeps   mysql（JDK版本信息）

解压Mysql

通常将软件安装到/usr/local

解压文件

tar   -xvf   mysql.tar   -C   目标路径（.tar是打包文件，不需要z）

安装客户端/服务端

rpm -ivh MySQL-server-5.6.22-1.e16.i686.rpm（注意一句话关于随机密码的文件位置）
rpm -ivh MySQL-client-5.6.22-1.e16.i686.rpm

启动服务

service mysql start

登录Mysql

mysql -uroot -p密码（在随机密码的文件里）

第一次访问需设置密码

set password=password(“密码”)

操作数据库

show databases

退出数据库

exit

远程连接

开放远程连接

grant all privileges on . to ‘root’ @'%’ identified by ‘root’;（显示Query OK则成功）

刷新

flush privileges;

如果远程连接失败，打开防火墙3306端口

/sbin/iptables -I INPUT -p tcp –dport 3306 -j ACCEPT
/etc/rc.d/init.d/iptables save
/etc/init.d/iptables status
学习阶段可以直接关闭防火墙
service iptables stop

安装redis（redis是源码，需要先编译）

redis是C语言开发，安装redis需要先将官网下载的源码进行编译，编译依赖gcc环境。

安装gcc-c++

yum install gcc-c++

安装redis

下载redis

wget http://download.redis.io/releases/redis-3.0.4.tar.gz

解压

tar -xzvf redis-3.0.4.tar.gz

编译安装

切换到程序目录，并执行make命令编译
make
执行安装命令
make PREFIX=/usr/local/redis install
make install安装完成后，会在/usr/local/bin目录下生成下面几个可执行文件，它们的作用分别是：
redis-server:Redis服务器端启动程序
redis-cli:Redis客户端操作工具。也可以用telnet根据其纯文本协议来操作
redis-benchmark:Redis性能测试工具
redis-check-aof:数据修复工具
redis-check-dump:检查导出工具

配置redis

复制配置文件到/usr/local/redis/bin目录： cp redis.conf /usr/local/redis/bin

启动redis

在当前目录下启动
./redis-server redis.conf

启动客户端

在当前目录下启动
./redis-cli
